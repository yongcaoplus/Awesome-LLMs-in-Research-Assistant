# 📚 Awesome LLMs in Research Assistant

![](./img/logo.png)

随着大语言模型（LLMs）在理解和处理科学文档方面表现出的强大能力，其作为研究助手的应用变得越来越具有前景。在本仓库中，我们旨在提供**关于使用LLMs作为研究助手的最新工作的全面总结**。此外，我们还包含该领域相关的平台、教程和研讨会等资源，帮助您快速开始相关项目。

欢迎任何形式的贡献和讨论！
